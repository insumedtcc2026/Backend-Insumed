Documentação oficial das rotas e controladores do sistema de pacientes.

Endpoints

  • GET /
  • GET /pacientesall
  • POST /pacientes
GET /

Retorna a página de documentação da API.

Respostas:

200 OK - Arquivo HTML renderizado com sucesso.

POST /pacientes

Cadastra um novo paciente no sistema. A senha enviada é criptografada antes de ser salva.

Corpo da Requisição (JSON Format):

{
  "pac_nome": "STRING" (VARCHAR (50)),
    "pac_email": "STRING" (VARCHAR(60)),
    "pac_telefone": "STRING" (VARCHAR(11)),
    "pac_cpf": "STRING"(VARCHAR(11)),
    "pac_data_nasc": "(AAAA-MM-DD)",
    "pac_sexo": "CHECK(1)" (M OU F),
    "pac_endereço": "STRING" (VARCHAR(100)),
    "pac_raça": "STRING" (VARCHAR(10))
}

Respostas:

201 Created 
{
  "message": "STRING"
}
500 Internal Server Error 
{
  "message": "STRING",
  "error": "STRING"
}
